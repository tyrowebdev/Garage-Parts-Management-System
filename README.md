# Garage Parts Management System
This project is based on java programming language.Java is used as backend for logic and JSP is used to handle server side logic.HTML and CSS is used for designing and styling. MySQL database is used with Apache Tomcat server. This project has a user module in which after logging in the user or admin can add new parts,view the existing parts details or can edit or delete them.He can also search for  a particular part by its respective id.
